
# Ink & Bean ☕📚

A multi-page web project built using **ASP.NET MVC** to practice layout design, page structuring, and basic separation of concerns using Controllers and Views.

---

## 📌 Project Overview
Ink & Bean is a 4-page website created to improve understanding of web application structure and consistent UI design across multiple pages.

---

## 🧱 Pages Included
- Home
- Menu
- Events
- Contact

---

## 🎯 Learning Objectives
- Working with **ASP.NET MVC structure (Controllers & Views)**
- Building reusable and consistent layouts
- Understanding page navigation flow
- Practicing front-end design integration within MVC

---

## ⚙️ Technologies Used
- ASP.NET MVC
- HTML5
- CSS3
